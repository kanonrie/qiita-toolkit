# TeX on Qiita

by @naoya_t

## これは何

Qiitaではてな的TeX記法を使えるようにするChrome拡張。

編集画面に &raquo;md, &raquo;tex というリンクが追加される。

## はてな的TeX記法 → markdown

```
[tex:e=mc^2]
```

>>md をクリック

```
![equation](http://chart.apis.google.com/chart?cht=tx&chl=e%3Dmc%5E2)
```

## markdown →はてな的TeX記法

```
![equation](http://chart.apis.google.com/chart?cht=tx&chl=e%3Dmc%5E2)
```

>>tex をクリック

```
[tex:e=mc^2]
```
